# SD Farmers Markets

A simple site showing which San Diego farmers markets are open today, this Saturday, and this Sunday.

**[samiprehn.github.io/sd_farmers_markets](https://samiprehn.github.io/sd_farmers_markets)**

## Features

- Today / Saturday / Sunday tabs (hides current day's tab if it's Saturday or Sunday)
- Open right now indicator with green ring
- Per-market weather at market midpoint hour (feels like temp, weather condition, UV index) — fetched per zip code using [Open-Meteo](https://open-meteo.com/)
- What's in season now, sourced from [Be Wise Ranch](https://www.bewiseranch.com/harvestschedule.htm)
- Dog friendly 🐾 indicator
- Parking notes
- Google Maps links for each market
- Markets grouped by region: Central, North County, East County, South Bay
